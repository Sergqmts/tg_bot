# VIBE Social Network

Социальная сеть на Flask + PostgreSQL + Tailwind CSS.  
Развёрнута на Railway: https://socnet.up.railway.app

**Ветка**: `main` (prod)

## Основные фичи
- Посты с медиа, лайки, реакции (emoji), комментарии, репосты, сохранения
- Личные и групповые чаты с голосовыми и видеосообщениями ("кружочки")
- **VoIP**: аудио/видеозвонки через WebRTC + WebSocket signaling + TURN
- **Shorts**: вертикальные видео с музыкой, лайки, комментарии, реакции
- **Stories**: 24h истории с фото/видео, реакции, комментарии
- **Музыкальный плеер**: Deezer API, плейлисты, избранное, рекомендации
- Сообщества (открытые/закрытые) с мероприятиями и архивами
- Уведомления (polling раз в 10с)
- Профили, подписки, блокировка, приватность, верификация телефона
- **Бот-платформа** (Telegram-style API, 25+ методов, вебхуки)
- **Фоторедактор** (11 инструментов: кадр, фильтры, эффекты, текст, стикеры, рисование, портрет, рамки, коллаж, анимация)
- **Видеоредактор для Shorts** (обрезка, фильтры, скорость, аудио — через Cloudinary)
- **Бизнес-аккаунты** и **мультиаккаунты**
- **Админ-панель** (модерация контента, жалобы, управление пользователями)
- **Система черновиков**
- **Google OAuth** вход
- **Модерация контента** (150+ NSFW-ключевых слов, авто-бан)

## Быстрый старт
```bash
python app.py  # dev :5000
```
Локально SQLite, продакшн PostgreSQL.
